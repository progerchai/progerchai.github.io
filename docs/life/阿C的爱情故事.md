# 阿 C 的爱情故事

我有一个朋友,这里我们叫他小 **C**,也就是故事的主人公,今天来聊一聊小 **C** 的爱情故事

小 **C** 身高不高,喜欢喝百香果的饮料,性格还行,社交还行,家里条件还行, 总之什么都过的还行,但在感情方面,却妥妥的是一个卑微仔,爱情的“舔狗”

主人公小 **C** 一共有两段感情经历,只不过两段都是失败的经历,他和同事饭后闲聊的时候开玩笑的说大概经历可以写一本"悲惨世界"了吧

## 故事 1

升到了初中,初三的时候,班级分班,机缘巧合,小 **C** 和 **L** 分到了一个小组,**L** 依旧是那么优秀,是班里的班长,学习好,人缘好

中考,小 **C** 考试失利,同样的 **L** 也考试失利,最终两个人去了两所不同的学校

分开总是失落的,小 **C** 心里的想法很少表露出来让人看见,有时候小 **C** 心里想如果那时候把想说的话说出来,但是一有这个念头,就胆怯了,不敢继续往下想

后来有一次,高二的时候,放学回家的公交车上,小 **C** 偶然遇到了 **L**,两者相见攀谈了起来最近的点滴,当 **L** 下车,小 **C** 看着背着书包的背影,心里的胆怯终于被打败,他觉得得试一把,说一说自己心里的想法

2016 年夏,天出奇的热,小 **C** 大一,志愿是姐姐填的,他自己没有废什么心力,原因是考砸了,实在是提不起兴趣做任何事,等反应过来的时候,志愿已经提交,连自己的志愿有哪些学校也不知道

临出发前,在学校边上的商务中心买东西,小 **C** 忽然就看见了 **T** 在买东西,心里自作多情地猜测是不是给自己买的,然后低头一看手机,就收到了来自 **T** 的消息,问几点走

和她说了之后,**T** 回复说“好,走之前等我一下”

临行前,室友帮小 **C** 的行李拿下楼,小 **C** 打好去火车站的车,等车马上到了,**T** 才急急忙忙地赶来,递给小 **C** 一个小袋子,这时候已经来不及说什么了,因为学区门口车很多,司机也等不了,小 **C** 半开玩笑地说“要不一起去火车站,送一送我吧”,没想到 **T** 仰头一想,说“好”

就这样,小 **C** 和 **T** 就一起去了火车站,打开 **T** 送的小袋子,里面是一些泡芙甜点之类的吃的,还有一杯 CoCo,小 **C** 最喜欢喝的百香果双响炮

小 **C** 进了车站,三步一回头,等走到了很里面的时候,回头发现 **T** 还在外面看着,于是拍了最后一张有关于她的照片,发消息给 **T** 让她回去吧,路上小心,到了学校记得回个消息

再后来就差不多是 20 年 7 月份了,**T** 回了云南老家,准备二战考研,而小 **C** 则是相信这 **T** 这次肯定可以考上心仪的大学

不知道是因为什么,可能是被小 **C** 打动了,也可能是其他的什么原因,有一天小 **C** 和 **T** 正在微信聊天突然怼了起来,当然是开玩笑的那种,然后 **T** 居然撒娇了起来,因为平常小 **C** 和 **T** 的聊天大多都是互怼的那种,搜索聊天记录一大堆“**T**M”那种,小 **C** 惊呆了,然后 **T** 莫名回了一句,“怎么,让一下自己女朋友不行吗”

就这样,莫名其妙了,小 **C** 和 **T** 就在一起了,虽然异地,但是也相处得很融洽,至于为什么,都不重要了

那天小 **C** 发了条朋友圈,“今天!我快乐了!”

再后来,没谈太久,小 **C** 和 **T** 分手,不清楚理由,一切都不清楚,小 **C** 就被分手了,可能是异地的原因吧,我不清楚

又一次小 **C** 喝酒喝多了,犯了傻,发了条短信给 **T** ,说“我想你了”,收到的回复是,拉黑,删好友,别再让联系了,我第一次见这么废物的男人

小 **C** 和 **T** 的感情,就像我写的这个故事一样,结束的很突然,很莫名奇妙,让人摸不着头脑

再后来,就没有后来了,也没听小 **C** 说过有关于 **T** 的任何事情

现在的小 **C** 是我最好的好朋友,还是喜欢喝百香果的饮料,工作还行,社交还行,总之什么都是还行,但在感情方面,他不想做太多但尝试了,他依旧相信爱情,只是现在但他有点害怕
